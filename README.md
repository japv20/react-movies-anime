# Movie & Anime Explorer

A React-based application built with Vite and Tailwind CSS that displays movies and upcoming anime using third-party APIs.

## Features
- Browse popular movies and upcoming anime.
- Interactive modals with synopsis, genres and release information.
- Responsive design across devices.
- Integration with multiple third-party APIs.

## Tech Stack
- React
- Vite
- Tailwind CSS
- JavaScript
- Rest APIs (TMBD & Jikan)

## What I learned
- Working with multiple APIs in a single React application.
- Handling API data safely using optional chaining.
- Styling using utility-first CSS with Tailwind.
- Debugging layout and positioning issues in modern frontend setups.
